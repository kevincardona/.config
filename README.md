<h1>setup:</h1>
~/.zshrc - add `source ~/.config/zsh/.zshrc`<br/>
~/.tmux.conf - add `source ~/.config/tmux/.tmux.conf`<br/>
nvim - needs to point to the nvim folder, should default to ~/.config/nvim<br/>
